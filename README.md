# 可乐商城系统
> 包含C端,B端和后端
---

## 📖 项目简介



## ✨ 功能特性

COLA ADMIN 后台管理系统 - 功能目录

**1. 数据看板**

功能概述：系统首页，集中展示核心业务数据、统计图表与关键指标。

**2. 商品管理**

***2.1 商品列表***
功能描述：查看、搜索、编辑、上架/下架所有商品。

***2.2 发布商品***

功能描述：新增商品，填写商品详情、规格、价格等信息。

***2.3 类目管理***

功能描述：管理商品分类（如一级、二级分类），设置分类属性。

***2.4 品牌管理***

功能描述：维护商品品牌信息，如品牌名称、Logo、简介。

**3. 订单管理**

***3.1 订单列表***

功能描述：查看所有订单，处理订单（确认、发货、退款），跟踪订单状态。

**3.2 售后客服**

功能描述：处理退货、换货、退款申请及客服工单。

**4. 用户管理**

***4.1 会员列表***

功能描述：管理注册用户/会员信息，查看用户资料、行为记录，可进行用户标签、等级管理。

**5. 内容管理**

***5.1 广告配置***

功能描述：管理网站/APP端的广告位、Banner图，设置广告内容与跳转链接。

***5.2 消息推送***

功能描述：向用户发送系统通知、营销短信或APP推送消息。

**6. 营销中心**

功能概述：管理促销工具，如优惠券、折扣活动、满减、秒杀、积分商城等营销功能。

**7. 库存管理**

功能概述：管理商品库存，包括库存查询、库存预警、入库/出库记录、盘点等功能。

**8. 系统设置**

功能概述：管理系统基础配置，如管理员账号与权限、操作日志、系统参数、支付/物流接口配置等。

## 🛠️ 技术栈

- **后端框架**: Spring Boot, Spring Security.
- **前端框架**: Vue 3,ElementPlus,Apache ECharts,AI Colors.
- **数据库**:  MySQL,Redis.
- **构建工具**:  Maven, Vite.

## 🚀 快速开始

### 环境要求

**后端要求**

- JDK >= 21
- MySQL >= 8.0
- Redis >= 5.0+
- Maven >= 3.8.9
- IDEA >= 21.0.8

**前端要求**

- node >= 22.20.0
- npm >= 8.0
- Vue >= 3.x
- Vite >= 4.x
